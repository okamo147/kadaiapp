e-Stat 人口統計可視化アプリ
政府統計ポータルサイト「e-Stat」のデータを活用し、日本の年齢別人口分布を可視化するStreamlitアプリケーションです。

1. 公開URL
https://kadaiapp-gjpv2x6qm8htxry5ztxhw6.streamlit.app/

2. 使用した統計データ
データ名: 人口統計 各年10月1日現在人口 令和２年国勢調査基準 統計表

出典: 政府統計の総合窓口(e-Stat)（https://www.e-stat.go.jp/）

ファイル名: FEH_00200524_260131160542.csv

データの内容: 各歳別、男女別の総人口（千人単位）

3. アプリの概要
このアプリは、複雑な構造を持つe-StatのCSVファイルを読み込み、指定した年次の年齢別人口推移を動的にグラフ化します。

主な機能
年次選択: サイドバーから調査年を選択し、即座にデータを切り替え

動的グラフ: 折れ線グラフとエリアチャートによる視覚的な分析

データ参照: フィルタリングされた詳細な数値をテーブル形式で確認可能

4. 使用したStreamlitのUI部品（3つ以上）
本アプリでは、課題要件を満たすために以下の部品を活用しています。

st.sidebar.selectbox: 膨大な調査年次から、ユーザーが閲覧したい年を直感的に選択させるために使用。

st.area_chart: 人口の「ボリューム」を視覚的に強調するため、標準の折れ線グラフに加えて採用。

st.expander: 画面をスッキリさせるため、詳細な生データ（テーブル）を折り畳んで表示。

st.line_chart: 年齢ごとの人口推移をシンプルかつ高速に描画。

5. 生成AIの活用状況:Yes
デプロイエラーの解消: クラウド環境（Streamlit Cloud）特有のライブラリエラーや日本語フォント問題を回避するため、Matplotlibに依存しない標準グラフへの切り替えを行った。
